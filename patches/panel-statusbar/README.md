# Panel Status Bar — Android Style

- Clock on the left side inside the indicator bar
- Clock background matches panel color (no bleed-through)
- Android-style quick tiles in dropdown (circular icons, labels, accent color)
- Identifier-based fallback icons (wifi, bluetooth, sound etc.)
- Network indicator hidden in minimized bar when no icon data
- Datetime indicator hidden from minimized bar (clock is on the left)
- Right inset for rounded display corners
- Left inset for clock area
